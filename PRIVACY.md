# Privacy Policy - AI Data Usage

## Data Collection
- Repository analysis data is processed locally and via Google Gemini AI
- No personal data is stored permanently
- API requests are proxied through secure backend

## AI Processing
- Code analysis is performed by Google Gemini Pro
- Data is transmitted securely via HTTPS
- No code is stored by the AI service beyond processing time

## Error Tracking
- Anonymous error data collected via Sentry (production only)
- No sensitive information (API keys, personal data) is logged
- Error data helps improve application stability

## Data Retention
- Analysis results stored locally in browser only
- No server-side data persistence
- Users can clear data anytime via browser settings

## Third-Party Services
- Google Gemini AI: Code analysis processing
- Sentry: Error monitoring (production)
- Vercel: Hosting and deployment

## Contact
For privacy concerns, please open an issue in the repository.
